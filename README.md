# Smart-Study-Copilot

Smart Study Copilot or simply Study Assistantis a simple Streamlit web app that transforms any study material into structured learning tools using the OpenAI API. I built this to help students study faster and effeciently.
You upload a PDF or paste your notes, and it can summarize them, make flashcards(cool), quiz you, explain concepts and more.

## Features

- PDF upload + text extraction
- Summary mode
- Flashcard with navigation.
- Quiz mode with scoring
- Chat mode (ask any question, it got you)
- Explain mode
- Trick quesrion mode

## AI declaration

- I used GitHub Copilot suggestions while writing the authentication module (auth.py). The OpenAI API (gpt-3.5-turbo and gpt-4o) powers all the study features — summarization, flashcards, quizzes, explanations, trick questions, and chat. The core app logic in app.py, the PDF extraction, and the quiz scorin, authentication were written by me without AI help.
